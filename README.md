# Django REST Framework Basics

## Title

Django REST Framework Basics

## Objectives

* To analyze and implement RESTful architecture using Django REST Framework (DRF) for building scalable web APIs.
* To design and develop API endpoints with serializers for structured data exchange.
* To apply and validate server-side data validation techniques including custom validators and file upload constraints.
* To integrate authentication and authorization mechanisms such as JWT in REST APIs.

## Theory

This experiment focuses on understanding the basic concepts of REST APIs using Django REST Framework. REST APIs allow communication between client and server using standard HTTP methods such as GET, POST, PUT, and DELETE. Django REST Framework provides tools such as serializers, views, and authentication mechanisms that simplify the process of building robust APIs.

## Procedure

The following tasks were performed during this experiment:

1. Parsing JSON data using Python and converting Python dictionaries back to JSON format.
2. Parsing XML data using Python and modifying XML structures.
3. Creating a server-side API for storing patient information with validation rules.
4. Designing a REST API for storing user data with specific validation constraints.
5. Developing an API to accept registration details and upload project files with defined validation rules.
6. Implementing an authentication API that verifies credentials and returns a JWT token.

## Discussion

In this experiment, various REST API development tasks were implemented using Django REST Framework. Different validation rules were applied to ensure data integrity. File upload constraints were tested to restrict file size and format. Authentication using JWT was also implemented to secure API access.
Some challenges encountered during the experiment included handling validation errors, configuring serializers correctly, and implementing secure authentication. These issues were resolved through proper configuration and testing.

## Conclusion

This experiment helped in understanding the fundamentals of REST API development using Django REST Framework. It provided practical knowledge of serializers, validation techniques, file handling and authentication mechanisms used in modern backend development.
